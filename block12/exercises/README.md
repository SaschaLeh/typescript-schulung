# Block 12: Quiz-App Exercises

Drei aufbauende Übungen, in denen du die Konzepte aus Block 9-11 im Kontext einer Quiz-Anwendung anwendest.

## Übungen

1. **Exercise 1: Interfaces & Type Guards** (Block 9)
   - Union Type für Antworttypen definieren
   - Question-Interface vervollständigen
   - Type Guards mit `typeof` implementieren
   - Type Narrowing in einer Funktion anwenden

2. **Exercise 2: Klassen & Generics** (Block 9 + 10)
   - Frage-Klassen implementieren (`checkAnswer`-Methoden)
   - Generische Hilfsfunktionen schreiben (`filterByCategory`, `findById`)

3. **Exercise 3: Error Handling** (Block 11)
   - Custom Error-Klassen erstellen (extends Error)
   - Type Guard für Fehlertypen implementieren
   - Sichere Funktion mit `throw` und `try/catch` schreiben

## Ausführen

```bash
npx ts-node block12/exercises/exercise1.ts
npx ts-node block12/exercises/exercise2.ts
npx ts-node block12/exercises/exercise3.ts
```

Entkommentiere die Testfunktion am Ende jeder Datei, um deine Lösung zu prüfen.

## Lösungen

Die Lösungen findest du im `solutions/`-Ordner:

```bash
npx ts-node block12/solutions/solution1.ts
npx ts-node block12/solutions/solution2.ts
npx ts-node block12/solutions/solution3.ts
```
